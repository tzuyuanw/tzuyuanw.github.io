Indego Bike Share Live Station Information
This page displays data on the bike share system in Philadelphia using data provided by Indego Bike Share. The live view shows current station status, including the number of bikes and docks available. The data displayed refreshes every 30 seconds.

Switching to the data view allows you to see data on trips made in January, February, and March of 2021.

The page was built using JavaScript and libraries such as Leaflet and Bootstrap.
