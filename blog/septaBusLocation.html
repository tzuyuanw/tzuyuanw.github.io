SEPTA Live Vehicle Information
This page visualizes SEPTA live vehicle data in the Philadelphia region. It includes all buses and trolleys currently in operation. The data displayed refreshes every 20 seconds.

Selecting a specific route allows the user to view all vehicles operating on the route, the routing, and stops.

The page was built using JavaScript and libraries such as Leaflet and Bootstrap. All data are obtained through SEPTA's publically available APIs.
